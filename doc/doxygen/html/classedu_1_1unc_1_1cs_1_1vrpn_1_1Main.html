◆ IDLE_INTERVAL

final int edu.unc.cs.vrpn.Main.IDLE_INTERVAL = 1000
static

The interval (in milliseconds) between calls to vrpn's mainloop() functions. Calling these on a timer prevents the client application from complaining that it hasn't heard from the Android server.
