1 // LatLonCoord.h: Definition of the CLatLonCoord class.
2 //
3 // The CLatLonCoord class, derived from the CTypedCoord class, represents a
4 // lat/lon coordinate. Since this class contains an internal lat/lon
5 // coordinate from CTypedCoord, no conversion functions are necessary.
6 //
7 // The formatted coordinate string generated by the CreateCoordString() member
8 // function can be in one of the three following formats:
9 // - Degrees (H DD.DDDDD*, H DDD.DDDDD*); this is the default.
10 // - Degrees, and decimal minutes (H DD*MM.MMM'. H DDD* MM.MMM')
11 // - Degrees, minutes, and decimal seconds (H DD*MM'SS.S", H DDD*MM'SS.S")
12 //
13 // The coordinate string format is set by calling the SetCurrentLatLonFormat()
14 // member function.
15 //
16 // Written by Jason Bevins in 1998. File is in the public domain.
17 //
