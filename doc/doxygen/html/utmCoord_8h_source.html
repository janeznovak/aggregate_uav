1 // UtmCoord.h: Definition of the CUtmCoord class.
2 //
3 // This class represents a coordinate on the UTM grid.
4 //
5 // Like all other typed coordinate classes, this class contains an internal
6 // lat/lon coordinate. This lat/lon coordinate is created by converting the
7 // lat/lon coordinate from a UTM coordinate. Since either lat/lon or UTM can
8 // be set and read, a conversion may be necessary from one type to another.
9 // This conversion is quite slow on machines without FPUs, so these conver-
10 // sions are done internally only when the converted coordinates are required
11 // by code using a UTM object (this is known as 'lazy evaluation.') In order
12 // to implement this lazy evaluation code, this class uses member functions
13 // to access the coordinates (since those member functions modify the lazy
14 // evaluation flags), rather than manipulating the coordinates directly.
15 //
16 // If the lat/lon coordinate contained within an object of this class is
17 // outside of the UTM grid boundary (>= 84 N or <= 80 S), the vertical UTM
18 // zone character is set to an asterisk ('*'). You can check the vertical
19 // UTM zone contained within this class to determine if the coordinate is
20 // outside of the UTM grid boundary.
21 //
22 // Written by Jason Bevins in 1998. File is in the public domain.
23 //
