242  // variables for transforming into texture coordinates
243  // problem this addresses: provide a continuous 2D->2D map from a texture plane onto
244  // the haptic plane where the map is a continuous function of the phantom
245  // position which minimizes local distortion. With such a map we can add
246  // texture to a curved surface so it feels almost like the texture on a
247  // flat plane as long as the texture scale is much smaller
248  // than the surface scale.
249 
250  // the parent node transforms the plane (0,1,0,0) into
251  // the current plane by incremental rotations about edges formed by
252  // consecutive planes - in this way we get a mapping that takes care of
253  // preserving orientation in the plane with each new plane update
254  // However, we are still left with a translation to maintain continuity
255  // between texture coordinates on one plane and the next and this is
256  // handled by updating a textureOrigin. textureOrigin is updated so that
257  // the projection of the phantom position onto the plane texture coordinates
258  // is preserved over plane updates
