Detailed Description

Class to support reading data from serial 5DT data gloves.

Parameters
nameName for the device
cConnection to use.
portserial port to connect to
baudBaud rate - 19200 for "wired"-type gloves (send/receive), 9600 implies a "wireless" (may be wired, but is send-only) glove
modeSet to 1 for the driver to request reports, set to 2 to stream them. (wireless implies 2, overriding value passed here)
tenbytesWhether reports should be 10 bytes instead of the documented 9. (wireless implies true, overriding value passed here)
