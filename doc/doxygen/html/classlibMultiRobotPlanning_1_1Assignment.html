Detailed Description

template<typename Agent, typename Task>
class libMultiRobotPlanning::Assignment< Agent, Task >

Find optimal (lowest total cost) assignment.

This class can find the lowest sum-of-cost assignment for given agents and tasks. The costs must be integers, the agents and tasks can be of any user-specified type.

This method is based on maximum flow formulation.
