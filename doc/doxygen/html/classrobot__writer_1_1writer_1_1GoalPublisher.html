Class constructor requires a namespace, x and y values (in meters) compared to the global origin and the rotation in radians.

With a rotation of 0.0 the robot is initially faced along the x axis, a rotation of 1.57 the robot is initially faced along the y axis. Declare a ROS 2 parameter backup_storage, when True store old action file into a backup folder.
