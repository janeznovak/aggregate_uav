# Installation notes

To install you have to use NodeJS version 14.14+.

## How to build
```bash
npm install --legacy-peer-deps
ng build
```

## How to run
Library usecase:
```bash
ng serve --configuration library
```

Hospital usecase:
```bash
ng serve --configuration hospital
```

All web components enabled:
```bash
ng serve
```

Angular app listens to: http://localhost:4200/
